# Reminder App

A fully functional **React Native reminder & task-management application** built using **Expo, Redux Toolkit, AsyncStorage, local notifications**, and custom theming.  
This app allows users to manage tasks, mark important items, track completed/incompleted tasks, and receive local reminders.
